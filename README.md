# המעבדה של מנדלייב – האטום (מודל בוהר)

רכיב מעבדת חקר אינטראקטיבי ללומדת מדעים לכיתה ח', במסגרת תוכנית "ישראל ריאלית".

התלמיד בונה אטומים על ידי הוספה והסרה של פרוטונים, נייטרונים ואלקטרונים, וצופה בזמן אמת
בלוח הנתונים המתעדכן. הייצוג החזותי הוא **מודל בוהר**: אלקטרונים הנעים סביב הגרעין ברמות
אנרגיה (קליפות). הרכיב הוא מעבדת חקר בלבד — ללא שאלות, בחנים או משוב הערכה.

## תכונות

- **אטום דינמי** — גרעין זוהר עם פרוטונים (אדום) ונייטרונים (אפור), ושתי רמות אנרגיה
  שעליהן מתמקמים האלקטרונים אוטומטית (עד 2 ברמה הפנימית, השאר בחיצונית).
- **לוח נתונים בזמן אמת** — שם היסוד וסמלו, מספר אטומי, מספר מסה, מטען חשמלי וזיהוי איזוטופ,
  עם כפתורי מידע (i) למספר מסה ולאיזוטופים.
- **שלוש משימות חקר** עם מד התקדמות לכל משימה (פרוטונים / נייטרונים / אלקטרונים).
- **כפתור סיום** שנפתח רק לאחר השלמת שלוש המשימות, עם הנחיה כמה משימות נותרו.
- **יסודות** — מימן עד חמצן (1–8). טווחים: פרוטונים 0–8, נייטרונים ואלקטרונים 0–10.

## הפעלה

קובץ HTML יחיד ועצמאי (תלוי רק בפונט מ-Google Fonts בעת חיבור לרשת). ניתן:
- לפתוח את `index.html` בלחיצה כפולה בדפדפן, **או**
- לצפות באתר דרך **GitHub Pages** (ראו למטה).

## GitHub Pages

להפעלת האתר: בריפו → **Settings → Pages → Branch: `main` / root → Save**.
לאחר מספר דקות האתר יהיה זמין בכתובת:

**https://nofarg8.github.io/Realistic-Israel-Periodic-Table/**

## חיבור ללומדה

עם סיום המעבדה (לחיצה על כפתור הסיום) הרכיב מודיע החוצה על השלמתו, כך שהלומדה
העוטפת יכולה לעבור לרכיב הבא:
- קריאה לפונקציה `window.onLabCompleted(detail)` — אם הוגדרה.
- שליחת אירוע `labCompleted` על `window` — ניתן להאזין עם `addEventListener`.

`detail` כולל `{ protons, neutrons, electrons }`.
